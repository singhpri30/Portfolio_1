Project Description

Eat-Da-Burger! is a restaurant app that lets users input the names of burgers they'd like to eat. It is a full stack application where data is storing in MYSQL database. Whenever a user submits a burger's name, the burger will be displayed in “Burger to devour” section, waiting to be devoured. When the user clicks on Devour it! button, the burger will move to the “Devoured” section. This application has been build using JavaScript, Handlebars, Express.js, Node.js and MYSQL.
